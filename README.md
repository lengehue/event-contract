# 📊 金融新闻情感分析系统

## 基于 BERT 的智能情感分类

---

<div style="text-align: center; margin-top: 100px;">

# 项目介绍

</div>

---

## 🎯 项目目标

构建一个**智能化的金融新闻情感分析系统**，能够：

- ✅ 自动识别新闻的**利好**或**利空**倾向
- ✅ 提供**高准确率**的情感分类
- ✅ 支持**批量处理**大量新闻
- ✅ 提供**友好的用户界面**和 **API 接口**

---

## 💡 应用场景

- 💰 **投资决策辅助**: 快速判断新闻对股价的影响
- 📰 **舆情监控**: 实时追踪市场情绪变化
- 🔍 **风险控制**: 提前识别潜在负面信息
- 📊 **数据分析**: 量化市场情绪趋势

---

---

## 📁 项目结构

## 🔧 核心技术栈

- **深度学习**: PyTorch + Transformers
- **预训练模型**: BERT (bert-base-chinese)
- **数据处理**: Pandas + NumPy
- **Web 服务**: Flask + Gradio
- **机器学习**: Scikit-learn

---

##  数据预处理流程

### 1️⃣ 数据清洗
- 去除 HTML 标签
- 去除 URL 链接
- 去除特殊字符
- 标准化空格

### 2️⃣ 标签生成
- 基于关键词规则
- 利好/利空分类

### 3️⃣ 数据集划分
- 训练集: 80%
- 验证集: 10%
- 测试集: 10%

---

## 🧠 模型架构

**关键参数**:
- 最大序列长度: 512
- 学习率: 2e-5
- 批次大小: 8
- 训练轮数: 5

---

## 🚀 训练流程

1. **加载预训练 BERT**
2. **添加分类头**
3. **冻结部分层**（可选）
4. **微调训练**
5. **验证评估**
6. **保存最佳模型**

**优化器**: AdamW  
**调度器**: Linear Warmup

---

## 🖥️ 用户界面

### Gradio Web 界面

- **单条分析**: 输入新闻，即时返回结果
- **批量分析**: 一次处理多条新闻
- **可视化**: 清晰展示分类结果和置信度
- **黑白配色**: 专业简洁的设计风格

---

## 🔌 API 接口

### RESTful API

- `GET /api/health` - 健康检查
- `POST /api/predict` - 单条预测
- `POST /api/predict_batch` - 批量预测

**特点**:
- JSON 格式请求/响应
- 支持批量处理（最多 100 条）
- 返回置信度评分

---

##  性能指标

训练完成后输出：

- ✅ **准确率** (Accuracy)
- ✅ **精确率** (Precision)
- ✅ **召回率** (Recall)
- ✅ **F1 分数** (F1-Score)

**分类报告**:
- 利好类别表现
- 利空类别表现
